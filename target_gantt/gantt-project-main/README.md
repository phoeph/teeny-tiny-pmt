# 甘特图项目管理工具

一个专业的甘特图项目管理工具，支持任务管理、时间规划和团队协作。

## 🌟 在线演示

- 首页直接展示项目导航与介绍

## ✨ 主要功能

### 🎯 智能任务管理
- **分层任务结构**：支持一级任务和二级子任务
- **智能分类系统**：根据任务名称自动分类（触达、孵育、转化）
- **时间范围设置**：可视化时间范围选择器
- **颜色预览**：实时颜色选择和预览

### 📅 灵活时间轴
- **双模式切换**：日模式和月模式自由切换
- **三级时间显示**：年-月-日三级时间轴
- **水平滚动**：支持时间轴和任务区域同步滚动
- **精确定位**：准确的时间计算和任务定位

### 👥 团队协作
- **负责人管理**：完善的人员分配和管理
- **任务编辑**：支持任务信息和时间范围编辑
- **状态追踪**：清晰的任务状态和进度显示

### 🎨 用户体验
- **现代化设计**：美观的渐变色彩和界面设计
- **拖拽操作**：支持任务条拖拽和大小调整
- **响应式布局**：适配各种屏幕尺寸
- **即时反馈**：操作结果即时显示

## 📋 使用说明

### 新增任务
1. 点击"新增任务"按钮创建一级任务
2. 点击任务行的"+"按钮创建二级任务
3. 填写任务信息和时间范围
4. 系统会根据任务名称自动分类

### 编辑任务
1. 点击任务的编辑按钮
2. 修改任务信息、时间范围或负责人
3. 支持实时颜色预览
4. 保存后任务条会自动更新位置

### 时间轴操作
1. 使用右上角按钮切换日/月模式
2. 点击"设置时间范围"调整项目周期
3. 使用鼠标滚动查看不同时间段
4. 拖拽任务条调整时间安排
